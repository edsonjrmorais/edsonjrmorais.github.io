How to Use These Charts

You can add this repository as a Helm repo using the following commands:

helm repo add my-charts https://edsonjrmorais/edsonjrmorais.github.io/helm-charts/
helm install my-release my-charts/my-chart

Make sure to replace my-release with the name of your release and my-chart with the actual chart name.
